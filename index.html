Setup needed

  1. Create a Google Cloud project and enable Google Calendar API.
  2. Create an OAuth 2.0 Web Client ID.
  3. Add your site URL (and http://localhost:8080) under Authorized JavaScript origins.
  4. Paste the Client ID into google-config.js.

Until that Client ID is set, Connect stays locked. No secrets belong in this frontend file except the public Client ID.
